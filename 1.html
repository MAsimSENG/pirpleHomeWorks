JavaScript Learning Sheet

Coding in JS

  1. <script> document.write("<p>This code from JS</p>"); </script>    ((Internal Code))
  2. <script src="main.js" > </script>    ((External File))

Where to put JS code

  1. Javascript code is put before body closing tag.
  2. It can be any where like between head tags or body tags but it is preferred to put it before the body closing tag, just in case that there's an issue in the Js code. As by doing so the rest of the page will run smoothly and is not interrupted because the Js code's crash.

JS Concatination

  • var x = 50,
        y = "Osama",
        z = x + y
    alert(z); --> 50Osama
  • var x = 50,
        y = "50",
        z = x + y
    alert(z); --> 5050
  • var x = 5,
        y = 10,
        m = "20",
        z = x + y + m
    alert(z); -->1520    ((5+10=15; 15 + "20" [The number converts to string 15 to "15"] = 1520))

JS Mathematical Operators

  • ( + ) Addition Operator.
  • ( - ) Substraction Operator.
  • ( * ) Multiplication Operator.
  • ( / ) Division Operator.
  • ( % ) Modulation Operator: Used to get the remainder of the devision.

JS Conditional Operators

  • ( = ) Assignment Operator : Used to assign a new value for the variable.
  • ( == ) Comparison Operator : Used to compare values in conditions.
  • ( === ) Identity Operator : Used to compare both values and data type in conditions.
  • ( !=) Not equal in value.
  • ( !==) Not equal in value OR in data type.
  • ( > ) Greater than.
  • ( < ) Less than.
  • ( >= ) Greater than OR Equal. (( notice the equal sign is on the right ))
  • ( <= ) Less than OR Equal. ((notice the equal sign in on the right ))

JS Logical Operators

  • ( ! ) NOT
  • ( && ) AND
  • ( || ) OR
